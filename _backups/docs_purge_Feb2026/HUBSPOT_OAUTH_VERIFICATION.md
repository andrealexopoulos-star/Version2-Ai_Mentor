- Option A: Merge.dev's HubSpot OAuth app configuration

## 🎯 REQUIRED ACTION (EXTERNAL TO BIQC)

**The user must configure HubSpot OAuth in Merge.dev:**

### Option 1: Use Merge's OAuth App (RECOMMENDED)
1. Login: https://app.merge.dev
2. Navigate: **Configuration** → **Integrations** → **HubSpot**
3. Select: **"Use Merge OAuth Application"**
4. Ensure Status: **Enabled**

### Option 2: Configure Custom OAuth App
1. In Merge dashboard, select: **"Use Custom OAuth Application"**
2. Get redirect URI from Merge (typically: `https://app.merge.dev/oauth/callback`)
3. In HubSpot dashboard:
   - Go to: **Settings** → **Integrations** → **Private Apps**
   - Create or edit OAuth app
   - Set **Redirect URI** to match Merge's callback
   - Set required scopes
4. Provide HubSpot Client ID and Secret to Merge
